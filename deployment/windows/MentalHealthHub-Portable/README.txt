MENTAL HEALTH HUB - PORTABLE EDITION
====================================

SYSTEM REQUIREMENTS:
- Windows 7 or later
- Node.js 14.0.0 or higher (Download from: https://nodejs.org/)

HOW TO RUN:
1. Make sure Node.js is installed on your system
2. Double-click START.bat
3. Open browser to http://localhost:3001

FEATURES:
- AI-powered mental health support
- Real-time anonymous chat
- Community forum
- Mental health resources

CONFIGURATION (Optional):
1. Copy .env.example to .env
2. Add your Hugging Face API key
3. Restart the application

TROUBLESHOOTING:
- If port 3001 is in use, close other applications
- Run as Administrator if you get permission errors
- Check firewall settings if you can't connect

For support: https://github.com/Soumyajit-M/mental-health-hub
