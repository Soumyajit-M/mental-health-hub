# Mental Health Hub - Standalone Application

## How to Run the Application

### Quick Start (Windows):
1. Double-click `START_APP.bat` 
2. Wait for the server to start
3. Open your browser and go to: **http://localhost:3001**

### Alternative Method:
1. Double-click `MentalHealthHub.exe` directly
2. Open your browser and go to: **http://localhost:3001**

## System Requirements
- Windows 7 or later (64-bit)
- 50 MB free disk space
- No Node.js installation required!

## Features
- ✅ Self-contained - No dependencies needed
- ✅ No installation required
- ✅ Runs on any Windows system
- ✅ AI-powered mental health support
- ✅ Real-time anonymous chat
- ✅ Community forum
- ✅ Mental health resources

## Configuration (Optional)
To use AI features with Hugging Face API:
1. Create a file named `.env` in the same folder as the .exe
2. Add this line: `HUGGINGFACE_API_KEY=your_api_key_here`
3. Restart the application

## Troubleshooting

**Port Already in Use:**
- The app runs on port 3001
- If you get an error, close any other programs using port 3001
- Or change the port in `.env` file: `PORT=3002`

**Firewall Warning:**
- Windows may ask for firewall permission
- Click "Allow access" to use the application

**Can't Connect:**
- Make sure you're accessing: http://localhost:3001
- Not https (no 's')
- Check if the console shows "Server running on port 3001"

## Stopping the Application
- Press `Ctrl+C` in the command window
- Or simply close the window
